## As a Library

### Maven

``` xml
<dependency>
    <groupId>com.github.jam01</groupId>
    <artifactId>xtrasonnet</artifactId>
    <version>0.3.0</version>
</dependency>
```

### Gradle
``` groovy
implementation "com.github.jam01:xtrasonnet:0.3.0"
```

### sbt
``` groovy
libraryDependencies += "com.github.jam01" % "xtrasonnet" % "0.3.0"
```
