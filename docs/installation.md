## As a Library

### Maven

``` xml
<dependency>
    <groupId>io.github.jam01</groupId>
    <artifactId>xtrasonnet</artifactId>
    <version>0.3.0</version>
</dependency>
```

### Gradle
``` groovy
implementation "io.github.jam01:xtrasonnet:0.3.0"
```

### sbt
``` groovy
libraryDependencies += "io.github.jam01" % "xtrasonnet" % "0.3.0"
```
