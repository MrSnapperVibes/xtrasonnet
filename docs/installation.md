## As a dependency

### Maven

``` xml
<dependency>
    <groupId>io.github.jam01</groupId>
    <artifactId>xtrasonnet</artifactId>
    <version>0.4.2</version>
</dependency>
```

### Gradle
``` groovy
implementation "io.github.jam01:xtrasonnet:0.4.2"
```

### sbt
``` groovy
libraryDependencies += "io.github.jam01" % "xtrasonnet" % "0.4.2"
```
