# xtr.url

## decode
`decode(data: String): String`

Returns the URL-decoded `data`.

**Example**
```
xtr.url.decode('Hello+World')
```
**Result**
```
'Hello World'
```

<br/>
## encode
`encode(data: String): String`

Returns the URL-encoded `data`.

**Example**
```
xtr.url.encode('Hello World')
```
**Result**
```
'Hello+World'
```
